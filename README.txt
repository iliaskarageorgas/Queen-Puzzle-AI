Για να τρέξετε το πρόγραμμα πρέπει να μεταφερθείτε μέσω Command Line στο directory 
που είναι αποθηκευμένα τα Queen_Puzzle.py, GeneticAlgorithm.py και Chromosome.py αρχεία.
Στη συνέχεια εκτελώντας την εντολή 'python Queen_Puzzle.py N P M S' το πρόγραμμα εκτελείται
(όπου Ν, P, S ακέραιοι αριθμοί και Μ δεκαδικός αριθμός και πιο συγκεκριμένα:
N ο αριθμός των βασιλισσών
P το πλήθος των χρωμοσωμάτων του πληθυσμού
Μ η πιθανότητα μετάλλαξης ενός γονιδίου
S ο μέγιστος αριθμός βημάτων)
